Detailed Description

Typed reference counted memory allocator.

These functions provide facilities for allocating dynamic memory which you can retrieve the type of via a call to r_mem_type(). This allows you to retrieve the real type of e.x. a void *.

This memory is also reference counted via calls to r_mem_ref() and r_mem_unref().
